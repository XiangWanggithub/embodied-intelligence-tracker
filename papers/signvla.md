# SignVLA: Gloss-Free Vision-Language-Action for Sign Language

**arXiv**: [2026-02-25](https://arxiv.org/abs/...)  
**团队**: -

## 核心亮点

- **创新**: 首个手语驱动的 VLA 框架
- **特点**: 不依赖 gloss（手语书面标记），直接视觉→动作
- **目标**: 包容性人机交互

## 与传统方法区别

传统方法：Sign → Gloss → Text → Action  
SignVLA：Sign Image → Semantic Instruction → Action

## 意义

- 探索非语言输入控制机器人的可能性
- 特殊人群交互
