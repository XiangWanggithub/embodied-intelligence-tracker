# Agentic AI: 架构、分类学与 LLM Agent 评估

**arXiv**: [2026-01-18](https://arxiv.org/abs/2601.12560)  
**团队**: -  
**重要度**: ⭐⭐  
**分类**: survey

## 一句话总结

提出统一的 Agent 分类框架（感知→大脑→规划→行动→工具→协作），覆盖从单 Agent 到多 Agent 层级系统。

## 核心分类

| 模块 | 说明 |
|------|------|
| Perception | 输入感知 |
| Brain | LLM 作为认知控制器 |
| Planning | 从线性推理到推理时计算 |
| Action | 从固定 API 到 MCP/原生计算机使用 |
| Tool Use | 工具调用 |
| Collaboration | 多 Agent 协作 |

## 覆盖领域

- 软件工程
- 科学发现
- 网页导航
- **具身机器人**

## 对我们的启发

- Agentic AI 的统一分类框架
- 从 fixed API → MCP 的工具使用演进
- 对具身机器人中 Agent 架构的分析
