# 2026 第 9 周精选 (02-24 ~ 03-02)

## 🔥 本周 Top 5

### 1. H-AIM ⭐⭐⭐ 【框架】
**方向**: 分层多机器人规划 | **发表于**: ICRA 2026 方向  
LLM + PDDL + Behavior Trees 三级联架构。成功率从 12% 飙到 55%。分层框架的标杆工作。  
→ [详细笔记](../papers/reasoning/h-aim.md)

### 2. FAEA ⭐⭐⭐ 【框架】
**方向**: LLM Agent 零示范控制 | **有代码**  
直接用 Claude Agent SDK 控制机器人，MetaWorld 成功率 96%，不需要任何 demo。颠覆性思路。  
→ [详细笔记](../papers/reasoning/faea.md)

### 3. EgoScale ⭐⭐⭐ 【模型】
**方向**: 灵巧手操作 | **团队**: UC Berkeley  
从大规模第一人称人类视频学习灵巧手操作。数据 scaling 的重要方向。  
→ [详细笔记](../papers/manipulation/egoscale.md)

### 4. Spatial AI Agents ⭐⭐⭐ 【综述】
**方向**: Spatial AI 综述 | 742 篇论文  
三轴分类法，关键发现：分层记忆、GNN-LLM 融合、World Models。  
→ [详细笔记](../papers/survey/spatial-ai-agents.md)

### 5. ABot-M0 ⭐⭐ 【模型】
**方向**: VLA Foundation Model | **团队**: ByteDance  
字节跳动 VLA，Action Manifold Learning 解决跨硬件异构。  
→ [详细笔记](../papers/vla/abot-m0.md)

## 📊 本周趋势

### 模型侧
- **数据 scaling** 仍是热点：EgoScale 探索从人类视频 scale
- **VLA 部署优化**：QVLA 探索量化
- **Benchmark 爆发**：RADAR、BiManiBench

### 框架侧
- **LLM Agent 直接做控制** 成为新范式（FAEA）
- **分层架构回归**：LLM + 形式化规划（PDDL）+ 反应式执行（BT）
- **多机器人协调** 需要新通信机制（CommCP）
- **World Models** 重要性持续上升
